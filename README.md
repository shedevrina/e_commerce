# Проект: ядро для интернет-магазина (e-commerce)

## Описание: 
Ядро для интернет-магазина. В дальнейшем для этого ядра возможно будет реализовать любой интерфейс — от сайта до телеграм-бота.
## Установка:
1. Клонируйте репозиторий: 
```
git clone https://github.com/shedevrina/e_commerce.git
```
2. Установите зависимости:
```
pip install -r requirements.txt
```

## Использование:
Запустив файл main.py из корневой директории

## Тестирование:
1. Проект покрыт тест-кейсами с фикстурами - модуль ./test/...
2. Сформирован отчет покрытия тестами в папку ./calc_col с помощью операции:
```
    coverage run -m pytest
    coverage html -d htmlcov
```

## Документация:
1. Класс BaseProduct. Абстрактный класс. Родительский для класса Product. Имеет абстрактный метод класса - создание нового объекта класса;
2. Класс Product. Наследник абстрактного класса BaseProduct. Родительский класс для классов: LawnGrass, Smartphone;
3. Класс MixinPrint. Миксин для класса Product при создании объекта класса, выводит в консоль информацию об объекте;
4. Классы LawnGrass и Smartphone. Классы-наследники от класса Product;
5. Класс Category. Класс категории товаров;
6. Класс IterCategory. Класс для итеративного вывода продуктов категории.
